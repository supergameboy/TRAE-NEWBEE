# L0 - 项目目标

## L0.1 文档概览
### L0.1.1 分批次读取全部文档
使用 Search Agent 分批次读取 docs 目录下的所有文档，避免一次性处理过多文档造成的信息过载。

**批次划分策略：**
- 批次 1：参考文档索引.md、项目架构说明文档.md
- 批次 2：核心功能设计文档（按模块分组）
- 批次 3：技术实现文档（前端/后端/数据库等）
- 批次 4：其他辅助文档

### L0.1.2 生成文档情况报告
每批次读取完成后，生成该批次的文档情况报告。

### L0.1.3 合并生成总文档概览报告
汇总所有批次的报告，生成完整的文档概览。
